# Ligas-Pok-mon
No universo Pokémon, as ligas Pokémon é a maior instituição de batalhas da região, para se tornar um campeão o treinador deve passar pelas seguintes etapas: Derrotar 8 lideres de ginásios para conseguir as insígnias, a elite dos quatro assim que conseguir as insígnias e desafiar quatro treinadores mais poderosos da região.
